# Hotel Reservation System

## Project Overview
This is a Java-based console application for managing hotel reservations. It allows users to browse hotels, select rooms, enter guest details, and process bookings with validation.

## How to Build and Run
1. Open the project in VS Code.
2. Navigate to `src/main/java/com/hotel/Main.java`.
3. Click the "Run" button or press F5.

## How to Execute Tests
1. Open the "Testing" tab in VS Code (Beaker icon).
2. Click the "Play" button to run all tests.
3. Alternatively, run `mvn test` in the terminal.
